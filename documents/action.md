# About
You can use this when Calculate or Replace string


## List of $ values
* $h - value.playerHp
* $l - loop count
* $p - phase
* $t - tick
* $T - real time
* $r - random: 0 ~ 1
* $s - random: -1 or 1

## Supported Math Functions
* \+
* \-
* \*
* /
* %
* ^
* log(a, x)
* sin(x)
* cos(x)
* tan(x)
